FrogFind! NG
The modern web for vintage computers

What is FrogFind! NG?

FrogFind! NG is a search engine designed to make the modern internet accessible to vintage computers. It strips away JavaScript, CSS, and heavy markup, delivering clean, simple HTML that works on browsers from the early 1990s onwards.

Fork Notice

FrogFind! NG is a fork and Python reimplementation of the original FrogFind! project by Sean (Action Retro).

The original FrogFind! is written in PHP and was created to enable classic Macintosh computers to browse the modern web. FrogFind! NG reimplements the same functionality in Python with added security hardening, caching, and production-ready deployment.

How does it work?

1. Search

Queries are sent to DuckDuckGo. Results are stripped of advertisements and reformatted as plain HTML 2.0, compatible with Netscape 1.x, Mosaic, Lynx, iCab, and similar browsers.

2. Article Reader

When you click a search result, the target page is fetched server-side and processed through Mozilla's Readability algorithm (the same engine that powers Firefox Reader Mode). The result is clean article text, with all links rewritten through FrogFind! NG so you can keep browsing.

3. Image Proxy

Images are fetched, resized to a maximum of 300x200 pixels, and compressed so they load quickly on slow connections.

Tested on

Netscape 1.1, Netscape 2.0, iCab 2.x, Lynx, Links, w3m, and any browser that understands HTML 2.0.
